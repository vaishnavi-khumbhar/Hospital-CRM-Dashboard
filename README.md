# 🏥 Hospital CRM Dashboard

A modern and responsive **Hospital CRM Dashboard** built using **React (Vite)** and **Bootstrap**.

---

## 🚀 Features

- 📊 Dashboard with analytics
- 👩‍⚕️ Doctor management
- 🧑‍🤝‍🧑 Patient management
- 📅 Appointment scheduling
- 💳 Billing system
- 💊 Pharmacy section
- 📈 Reports & analytics
- ⚙️ Settings panel
- 🎬 Smooth animations using Framer Motion
- 📱 Responsive sidebar (mobile-friendly)

---

## 🛠️ Tech Stack

- React (Vite)
- React Router DOM
- Bootstrap 5
- Framer Motion
- React Icons
